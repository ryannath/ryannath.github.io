HTML

Introduction

  • HTML is the standard markup language for Web pages
  • HTML stands for Hyper Text Markup Language
  • It's basically a way of organising content of a page, labelling them with specific tags

The following is an example of a html code. You can see how a browser renders it by following this link: html_example1.html

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
</html>

The indentation is unnecessary, I just added it in to make it look better. You may also noticed that html elements can be nested, i.e. the <head> tag is inside the <html> tag

Specifics

So as we can see before, html consists of these sort of tags surrounded by angle brackets <>. Normally, html tags has the following layout:

<tagname> content goes here... </tagname>

So the tags normally comes in pairs, with the first one being called an opening or start tag and the second one being called a closing or end tag. The end tag is indicated by the forward slash inserted before the tagname.

The tags of the previous example:

  • The <!DOCTYPE html> declaration defines this document to be HTML 5
  • The <html> element is the root element of a HTML page <- what start the html document itself
  • The <head> element contains meta information about the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph.

Also note that only the things in between the body tags are actually displayed. Metadata of the document is not displayed and is used for other purposes. This includes the title tag. However, the title tag is displayed as the name of the tab in a browser. But yea, it is not shown on the actual page itself.

HTML Basic

Here introduced some basic html tags.

Headings

You have seen <h1>, but there are also smaller headings available, such as <h2>, <h3>, all the way to <h6>.

Links

HTML links are defined with the <a> tag, like so:

<a href="http://www.google.com">This is a link</a>

The destination is specified within the href attribute, attributes will be discussed later. The actual thing in between the tags is the words that you can click to go to the specified href link. The name href stands for hypertext reference

Images

HTML images are defined with the <img> tag, note that <img> is an empty element and could also be written like so <img />. Empty elements are also called void elements, they are elements that must not contain any content (the thing in between the tags) and do not have a closing tag.

<img src="https://assets.jpegmini.com/user/images/slider_puffin_jpegmini_mobile.jpg" alt="An image of a bird" width=400 height=400>

This will show an image like the one below. The src attribute gives the path to the photo, this can be a url or a local path. The alternative text is the text shown if the image could not be found/displayed. It is also useful for those who is not reading the webpage, instead using text to speech technology, etc. The width and heigh property specifies the size the image should be displayed in, in this case 400 pixels by 400 pixels. You can also align this photo by adding the style attribute. The ellipsis is not code. It's just used here so the code block is not too long.

<img ... style="display: block; margin-left: auto; margin-right: auto">

This is looked into more deeply later with CSS.


Note that you don't need to put a comma between different attributes.

To use a local path, you must make sure that the image actually exist, so it will look like the one below, assuming a file called bird_image.jpg exist in the current directory.

<img src="bird_image.jpg" alt="An image of a bird" width=400 height=400>

HTML Buttons

HTML buttons are defined with the <button> tag.

<button>Click me</button>

It will show it like this: with the text in between being the text on the button

HTML Lists

HTML lists are defined with the <ul> (unoreder/bullet list) or the <ol> (unordered/numbered list) tag, then followed by <li> (list element) tags:

<ul>
    <li>Coffee</li>
    <li>Tead</li>
    <li>Milk</li>
</ul>

<ol>
    <li>Coffee<li>
    <li>Tead</li>
    <li>Milk</li>
</ol>

This would look like this:

  • Coffee
  • Tea
  • Milk
  1. Coffee
  2. Tea
  3. Milk
